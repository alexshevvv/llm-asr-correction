# LLM-ASR-Correction
# DEMO (IN DEVELOPMENT)
Дипломная работа: **Распознавание речи с корректировкой ошибок с использованием больших языковых моделей**

## Описание

Исследование эффективности использования LLM (GPT-4, Claude) для постобработки и коррекции ошибок в выходных данных систем автоматического распознавания речи (ASR).

## Задачи

1. Найти открытые модели ASR (Whisper, GigaAM) - Статус: выполнено
2. Найти датасеты русской/английской речи - Статус: выполнено
3. Запустить модели, измерить WER - Статус: в работе
4. Использовать LLM для коррекции ошибок, пересчитать WER - Статус: в работе

## Модели ASR

| Модель | Язык | Описание |
|--------|------|----------|
| Whisper large-v3 | RU/EN | OpenAI, SOTA multilingual |
| GigaAM-v3 | RU | SberDevices, SOTA для русского |

## Датасеты

### Русский язык
- **Golos** (~1240 часов) — Сбер, разнообразная речь
- **Russian LibriSpeech** (~98 часов) — аудиокниги
- **Common Voice RU** (~200+ часов) — краудсорсинг

### Английский язык
- **LibriSpeech** (~960 часов) — аудиокниги
- **Common Voice EN** (~2000+ часов) — краудсорсинг
